Angular Tiny Calendar

A (very) small calendar directive, which can also handle events

Installation

For all information about installation visit the github repo.

Usage

Basic

Basic usage with default options (English language, first day is Sunday)

<div tiny-calendar></div>

Start on Monday

Changing first day (0 = Sunday, 1 = Monday â€¦)

<div tiny-calendar first-day="1"></div>

Localization

Changing first day, setting custom labels for days and "today" button

<div tiny-calendar first-day="1" week-days='DLMMGVS' today="oggi"></div>

Adding events

Every event *must* have a title and a startDate. It is also possible to add the time of the event and the endDate

<div tiny-calendar events="events"></div>
        
events: {{eventsDisplay}}
